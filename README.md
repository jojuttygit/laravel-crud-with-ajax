
## Employee CRUD operations with Laravel 8, Bootstrap 4 and AJAX with file upload.

## Requirements

npm 8.0.0

## Package

sweetalert2

## Operations

- Create an employee
- View employees
- Edit and update an employee
- Delete an employee
- Bulk delete

Please run below commands once the repo is cloned,

- composer install
- npm install
- npm run dev
- php artisan migrate
- php artisan db:seed
- php artisan storage:link
